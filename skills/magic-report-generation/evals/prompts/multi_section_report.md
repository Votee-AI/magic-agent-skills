# Eval: Multi-Section Report

## Task

You have analysis results from a customer churn study. The findings include statistical analysis, data profiling results, and 5 visualizations (saved as PNG files). Generate a report for an **executive audience** that also includes a summary data table of the top 10 churning customer segments.

The findings JSON is at `output/churn_findings.json`. The charts are at `output/charts/`. Save the report to `output/churn_report.md`.

## Context

- Audience is C-suite executives -- they want decisions, not methodology
- Historical baseline exists: last quarter's churn was 8%, industry average is 15%
- Current churn rate is 12% (95% CI: 10.5-13.5%, N=15,000 customers)
- A segment summary DataFrame exists at `output/segment_summary.csv` with 45 rows and 8 columns
- 5 charts have been generated by magic-data-visualization
