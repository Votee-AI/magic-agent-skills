# Magic Data Exploration

Explore data interactively and detect patterns systematically — follow threads freely or run scripted segment and relationship analysis.

## What This Skill Does

- Detects patterns automatically across columns (value templates, frequency clusters, structural similarities)
- Compares segments and groups to surface differences, anomalies, and correlations
- Explores relationships between columns including non-linear dependencies and co-occurrence structures

## Files

- `SKILL.md` — Agent knowledge document and frontmatter
- `scripts/` — `detect_patterns.py`, `prepare_for_exploration.py`, `segment_analysis.py`, `relationship_explorer.py`

## Related Skills

- `magic-data-profiling` — profiling gives the quality overview; exploration goes deeper
- `magic-statistical-analysis` — move to this skill when exploration surfaces a hypothesis to test
- `magic-data-visualization` — visualize findings discovered during exploration
