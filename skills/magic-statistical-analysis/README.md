# Magic Statistical Analysis

Perform descriptive statistics, hypothesis testing, and correlation analysis with mandatory uncertainty communication and effect-size reporting.

## What This Skill Does

- Computes descriptive statistics with distribution-appropriate summaries and normality testing (Shapiro-Wilk)
- Runs hypothesis tests (t-test, chi-square, Mann-Whitney, ANOVA) with p-values, confidence intervals, and effect sizes
- Performs correlation analysis with method selection (Pearson for normal data, Spearman for non-normal)

## Files

- `SKILL.md` — Agent knowledge document and frontmatter
- `scripts/` — `descriptive_stats.py`, `hypothesis_test.py`, `correlation_analysis.py`

## Related Skills

- `magic-data-profiling` — profiling precedes statistical analysis to validate data quality
- `magic-data-exploration` — exploration surfaces hypotheses; statistical analysis tests them
- `magic-data-visualization` — visualize distributions and test results for reporting
